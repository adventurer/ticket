# ticket
腾讯的面试题交稿（并发买票）


端口：8088 </br>
买票接口：/ticket?c=\d  (数字1-5表示买1张或5张)</br>
检票接口：/check</br>
例：</br>
"35bc3449-5d84-4514-bbca-81691b04390a":[829]</br>
